# WashHub

Landing page sederhana + konfigurasi PHP/MySQL untuk XAMPP.

## Menjalankan

1. Start Apache dan MySQL dari XAMPP Control Panel.
2. Buka `http://localhost/washhub/` untuk melihat landing page.
3. (Opsional) Buka `http://localhost/washhub/health.php` untuk cek koneksi database.

## Database

- Host: `localhost`
- DB: `washhub_db`
- User: `root`
- Password: (kosong)

Bila password root Anda berbeda, ubah di `config.php`.
